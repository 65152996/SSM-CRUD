# SSM-CRUD
### 简介
基于SSM框架的增删改查小系统，完成了基本的增删改查业务，涵盖大量知识点。
### 涉及的知识点
三大框架：Spring + SpringMVC + Mybatis <br>
前端框架及知识点：Bootstrap + Ajax+js <br>
其他：Mybatis Generator + PageHelper（分页插件） <br>
